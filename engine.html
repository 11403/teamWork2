-- Engine --




- Unity -

Games Pros
  • Rust
  • Tarkov
  • Subnautica
  • Easy to Use
  • Best for Beginners

Unity is the first game engine any developer should start on. It is the easiest to learn and is compatible with all platforms. Unity also has tutorials that help teach you how to use the engine. Graphics will require more effort than other engines to look good.

- Unreal Engine -

Games Pros
  • Fortnite
  • Star Wars: Fallen Order
  • Sea of Theives
  • Graphical Capabilities
  • Easy to Use
  • Platform Compatibility

Unreal Engine is one of the most popular game engines among triple A developers. It's graphics combined with ease of use and Compatibility make it the best choice for experienced develpers.

- Frostbite -

Games Pros
  • Battlefield 2042
  • Madden NFL 22
  • Star Wars Battlefront 2
  • Great Graphics
  • Difficult to Use

Frostbite is a game engine developed by EA, almost all of their games run on the Frostbite engine. It is relatively hard to use and learn but it produces amazing graphics and gameplay physics.
